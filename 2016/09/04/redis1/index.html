#redis的数据类型

打开redis数据库：

##strings:
set key=values
例如： set name=shlzx
获取name的对应值： get name

#####setnx

如果key已经存在，则返回一个0
作用：防止设置出现覆盖现象

nx 代表 not exsite

#####setex

setex 设定值的有效值，若不设定有效值则永久有效
例如： setex name 10
有效期为10秒
setrange 设定values 的子字符串

#####mset:

设置多个key值

#####mget

同时获取多个key值

incrby 自增 自减 可以自定义
